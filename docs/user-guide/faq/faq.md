# FAQ (자주 묻는 질문)

---

## 1. 감정 분석 결과가 이상해요. {#unexpected-emotion-result}
가끔은 결과가 지금의 마음과 다르게 느껴질 수 있어요.  
다정문장은 입력한 문장의 표현 방식과 맥락을 바탕으로 감정을 분석하기 때문에,  
짧거나 애매한 표현일수록 의도와 다르게 해석될 수 있습니다.

### 이렇게 시도해 보세요:

* 감정을 조금 더 구체적으로 적어보세요.  
  * 예) 힘들다 → 하루 종일 긴장해서 많이 지쳤어요
* 상황이나 이유를 한 문장 덧붙여 주세요.
* 여러 감정이 섞여 있다면, 가장 크게 느껴지는 감정을 중심으로 적어보세요

## 2. 감정 분석을 받기 위해 작성한 내용도 저장되나요?
감정 분석을 위해 입력한 문장은 분석 후 즉시 처리되며 저장되지 않습니다.  
다정문장은 사용자의 사적인 감정을 존중합니다.  
오직 사용자가 직접 필사하고 저장한 기록만이 개인 기록으로 안전하게 보관됩니다.

## 3. 위로 문장이 계속 비슷하게 나와요.
비슷한 감정과 상황을 반복해서 입력하면 유사한 문장이 추천될 수 있어요.

**조금 다른 위로를 받아보고 싶다면:**

* 표현 방식을 살짝 바꿔보세요.
* 감정의 이유나 상황을 덧붙여 보세요 하루의 다른 순간을 떠올려 적어보세요.

**같은 감정도, 다른 말로 적으면 다른 위로를 만날 수 있어요 🌱**

## 4. 필사가 저장되지 않았어요.
필사 내용은 완료 후 자동 저장됩니다.

**아래 사항을 확인해 주세요:**
* 필사 문장을 끝까지 따라썼는지
* 네트워크 연결이 안정적인지

## 5. 기록이 사라졌어요.
**다음 항목을 먼저 확인해 주세요:**
* 로그인한 계정이 이전과 동일한지
* 필터(기간 설정)가 적용되어 있지는 않은지
* 네트워크 상태가 안정적인지
