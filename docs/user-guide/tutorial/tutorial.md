# 시작하기

다정문장은 복잡한 설정 없이  
**지금의 마음을 적는 것만으로 바로 시작**할 수 있어요.  
아래 순서에 따라 천천히 이용해 보세요. 

---

## 1. 무료로 시작하기 🌿

다정문장 웹 서비스에 접속합니다.  
**[무료로 시작하기]** 버튼을 클릭하면 로그인 페이지로 이동합니다.

![첫 화면](/img/dajeong_tutorial_01.png)

![로그인 화면](/img/dajeong_tutorial_02.png)

## 2. 오늘의 감정 기록하기 📝

로그인을 완료한 후,  
화면에 보이는 입력창에 지금의 기분이나 하루를 짧은 문장으로 적어주세요.  
길지 않아도 괜찮아요. 솔직한 한 문장만이면 충분합니다. 👍

입력을 마치고 **[위로의 문장 밭기]** 버튼을 클릭하면  
감정 분석이 자동으로 시작됩니다.

![감정 입력 화면](/img/dajeong_tutorial_03.png)


## 3. 위로 문장 필사하기 ✍️

AI가 입력한 문장을 바탕으로 현재의 감정을 알려줍니다.  
결과가 예상과 다르게 느껴질 수도 있어요.  
👉 이 경우 **[감정 분석 결과가 예상과 다를 때](../faq#unexpected-emotion-result)** 를 참고해 주세요.

당신의 마음을 위로해주는 문장을 천천히 따라 써보세요.  
필사를 마치면 기록은 자동으로 저장됩니다.

![결과 화면](/img/dajeong_tutorial_04.png)
![팝업창](/img/dajeong_tutorial_05.png)

## 4. 기록 돌아보기 📖

주간 / 월간 / 연간 단위로  
이전에 작성한 필사 기록을 확인할 수 있습니다.

감정의 흐름을 되돌아보며  
나의 변화를 천천히 느껴보세요. 🌙

![라이브러리 화면](/img/dajeong_tutorial_06.png)
  
> **누군가에게 보이기 위한 글이 아니라**  
> **나를 위한 조용한 기록이에요 🤍**
